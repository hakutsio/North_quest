# 🧭 North Quest

> Um jogo educacional desenvolvido em *Godot 4*, com temática medieval, criado para auxiliar o ensino de Geografia por meio da exploração e da orientação espacial.

# 📖 Sobre o Projeto

O objetivo é ensinar conceitos de orientação espacial de forma interativa, utilizando uma aventura medieval em mundo aberto.

Durante a jornada, o jogador aprende:

- 🧭 Utilização da bússola;
- 🌎 Pontos cardeais;
- 📍 Orientação por pontos de referência;
- 🗺️ Localização espacial;
- 🎮 Exploração de mapas.

O jogo foi desenvolvido utilizando a engine *Godot 4*, buscando unir diversão e aprendizado.

Durante a aventura o jogador poderá:

- Explorar o mapa;
- Coletar itens;
- Utilizar a bússola;
- Aprender conceitos de Geografia;
- Completar desafios educacionais.

---

# 📚 Objetivo Educacional

O jogo foi desenvolvido para apoiar o ensino da disciplina de Geografia no Ensino Fundamental II.

Conteúdos abordados:

- Orientação espacial;
- Pontos cardeais;
- Localização geográfica;
- Interpretação de mapas;
- Utilização da bússola.
